# Enterprise Data Warehouse
This repository is used as personal project for data warehouse in PostgreSQL using pg_mooncake.

## Architecture Overview
This data warehouse implements medallion architecture for data warehouse with bronze functions as raw data storage, silver layer for cleansing and validating data and gold layer for 
presenting to users.
